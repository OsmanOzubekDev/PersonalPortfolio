Portfolio Website
=================

This repository contains a clean and responsive personal portfolio website built with HTML, CSS, and JavaScript. It is designed to showcase professional experience, projects, skills, and contact information in a modern and mobile-friendly layout.

Features
--------

- Fully responsive design for desktop and mobile
- Portfolio project showcase section
- Skills and services overview
- About section with personal introduction
- Contact form and social links integration
- Easy to customize layout and colors

How to Use
----------

1. Open `index.html` in your browser to view the site locally.
2. Update the content in `index.html` to match your experience and projects.
3. Customize styles in `css/styles.css` and add your own images in `images/`.
4. If you need JavaScript interactions, edit `js/main.js`.

Customization
-------------

- Change text and headings directly in `index.html`
- Replace portfolio images in `images/portfolio/`
- Adjust colors and spacing in `css/styles.css`
- Add or remove sections as needed to fit your personal brand
